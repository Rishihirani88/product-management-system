# Product Management System (Python + MySQL)

## Description
This is a command-line based Product Management System developed using Python and MySQL. It allows users to manage product records efficiently.

## Features
- Add product
- Update product
- Delete product
- View all products
- Search product by ID

## Tech Stack
- Python
- MySQL
- Tabulate Library

## How to Run
1. Install MySQL and create a database named 'project'
2. Create a table 'product'
3. Install required libraries:
   pip install mysql-connector-python tabulate
4. Run the program:
   python product_management_system.py
